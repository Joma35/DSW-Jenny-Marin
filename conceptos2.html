Conceptos CSS

Un Poco De Historia

Cascading Style Sheet (CSS) , que traducido nos dice Hojas de estilo en cascada , las hojas de estilo han evolucionado desde sus primeros estándar en los años 80, se desarrollo CSS debido que se requería proveer estilos en diferentes estilos web. los existentes en aquella época, como DSSL y FOSI no suplían las necesidades. Html cada vez , tenía más evolución y las hojas de estilo se les dificultaba tener el control sobre como mostrar el contenido web. Se presentaron 9 diferentes lenguajes de hojas de estilo, de estas dos propuestas influenciaron en lo que sería CSS y fueron: Cacading HTML Style y Stream-base Style Sheet Proposal (SSP) de ahí se fusionó el trabajo de estos dos lenguajes y se desarrollo lo que conocemos como CSS Nota: La 'H' se eliminó del nombre porque estas hojas , pueden ser aplicadas a otros lenguajes de marcado.
